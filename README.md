# srt-merge-toolプロジェクト

## 概要
日本語字幕（基準）と英語字幕を時間情報をもとにsrtファイルのテキストを結合する Python スクリプトです。  
英語字幕のうち、日本語字幕と時間がずれているものも条件に応じて保持します。
動画上に同時に表示させるようにします。

- 日本語字幕（SRT）を基準に結合
- 英語字幕が
  - 次の字幕と時間的に近い／重なる場合  
    → 次の英語字幕の先頭に結合
  - 時間的に大きくずれている場合  
    → 英語字幕のみで保持
- UTF-8 対応
- 外部ライブラリ不要（標準 Python のみ）

## 使い方
- 手順1 以下の3ファイルを同じフォルダに置きます。

- `jimakumerge.py`
- 日本語字幕ファイルを用意しこのファイル名にする（japanese.srt）
- 英語字幕ファイルを用意しこのファイル名にする（english.srt）

※ 字幕ファイルは **UTF-8** に変換してください。
- 手順2 プログラムを起動しファイルのテキストを結合させる。
-         ```bat
          py jimakumerge.py
- 手順3 出力
  結合された字幕ファイルが同じフォルダに生成されます。


## 必要な環境
